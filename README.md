# Odin-Recipe

Project: Landing Page

Description
This is a landing page project for The Odin Project's Foundations course. The goal was to build a complete web page from a provided design, practicing HTML structure and CSS layout techniques from scratch.

Skills Demonstrated
Flexbox Mastery: Used display: flex to align the header, hero content, information boxes, and the call-to-action banner.
CSS Organization: Managed all styling within a single stylesheet as recommended by the project guidelines.
Semantic HTML: Built a clear structure using sections and appropriate tags to organize the 4 main sections and a footer.
Design Implementation: Replicated a professional layout focusing on color contrast, font weights, and spacing.
